## How to Use

1. **Select a Category** - Choose from cardiovascular, infections, neurology, metabolic, pain, stability, or view all drugs
2. **Study the Front** - Review the generic name, brand name, drug class, and primary indication
3. **Flip the Card** - Tap "Tap to Flip" to see side effects, adverse reactions, and storage information
4. **Navigate** - Use Previous/Next buttons to move through the deck
5. **Track Progress** - Progress bar shows your position in the current deck
